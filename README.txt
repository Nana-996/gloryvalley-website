# Glory Valley School Website

## Files
- index.html  → Home page
- join.html   → Enrollment/Join form
- style.css   → All styles
- logo.jpeg   → School logo (add your own)

## Deployment Steps
1. Download and unzip these files
2. Add your logo.jpeg into the same folder
3. Create a free GitHub account → github.com
4. Create a new repository (e.g. gloryvalley-website)
5. Upload all files to the repository
6. Go to netlify.com → "Add new site" → "Import from Git"
7. Connect GitHub → select your repo → click Deploy
8. Your site will be live at yoursite.netlify.app
9. To see form submissions: Netlify Dashboard → Forms
